# PDF-Knowledge-Base

üöÄ RAG-powered intelligent assistant using DeepSeek-R1 & Sentence Transformers

This project allows users to upload a PDF, build a knowledge base using embeddings, and ask context-aware questions.
It uses Retrieval-Augmented Generation (RAG) with DeepSeek-R1-1.5B for accurate, clean, to-the-point answers.
